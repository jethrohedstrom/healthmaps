# Spacing Rules

- Generous whitespace is the #1 design priority — when in doubt, add more space
- Hero sections: content left-aligned, generous vertical padding (pt-20 pb-8 md:pt-32 md:pb-12)
- Section-to-section: py-24 md:py-32 (homepage), py-16 md:py-24 (inner pages)
- Page container: max-w-6xl (1280px) with horizontal padding
- Body text max-width: 720px (max-w-3xl)
- Card grid max-width: 960px
- Feature card padding: p-12 (48px)
- Feature card grid gap: gap-12 (48px)
- Reference sites for spacing feel: nhn.no, app.oevra.com
